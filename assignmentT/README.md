## 2. 代码说明

* 前端位于 `./frontend`

  * 构建：`npm install`

  * 运行：`npm start`

  * 基于 React + Ant Design

  * **预编译版本位于** `./frontend/build`

    * 在 `./frontend` 目录下执行

      ```bash
      npm install -g serve
      serve -s ./build
      ```

    * 即可启动前端

* 后端位于 `./backend`

  * 构建：`npm install`
  * 运行：`npm start` 或 `node ./index.js`
  * 使用 **express.js** 实现的一个极端轻量化的Redis风格后端
  * 后端无预编译版本（因为本身就只有单文件）
